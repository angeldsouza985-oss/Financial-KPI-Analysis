# Financial KPI Analysis Dashboard

## Project Overview

This project analyzes the financial performance of a startup using Python and Power BI. It calculates important business KPIs and presents them through an interactive dashboard.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

## KPIs

- Total Revenue
- Total Expenses
- Total Profit
- Profit Margin
- Customer Acquisition Cost (CAC)
- Customer Lifetime Value (LTV)
- LTV:CAC Ratio

## Dashboard Features

- Revenue Trend
- Profit Trend
- Revenue vs Expenses
- Customer Growth
- Profit Margin Analysis
- LTV vs CAC Comparison
- Month Filter
